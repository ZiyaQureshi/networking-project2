# Secure Networking Website

## Project Overview
This project is a secure, optimized, and monitored website created for a Computer Networking assignment. The website demonstrates HTTPS deployment, basic protection against XSS attacks, performance optimization, and traffic monitoring.

## Features
- HTTPS enabled through GitHub Pages
- User input sanitization using DOMPurify
- Content Security Policy added in HTML
- Lazy loading image optimization
- Mobile-friendly responsive design
- Google Analytics support for traffic monitoring

## Security Enhancements
1. HTTPS is enabled automatically by GitHub Pages.
2. DOMPurify is used to sanitize user input and reduce the risk of XSS attacks.
3. A Content Security Policy is included to limit unsafe scripts.

## Performance Enhancements
1. Images use lazy loading.
2. CSS and JavaScript are separated.
3. The website uses a lightweight responsive design.

## Monitoring
Google Analytics can be added by replacing the placeholder Measurement ID in index.html.

## Deployment Instructions
1. Upload all files to a GitHub repository.
2. Go to Settings.
3. Click Pages.
4. Select Deploy from branch.
5. Choose main branch and root folder.
6. Save and wait for GitHub Pages to publish the site.
